# datareduction-task
an assignment in datareduction

In swedish:
•Välja vilket dataset från öppna källor (egen data) som ni vill och undersöka flera variabler i datasetet
•Köra statistik på dessa variabler
•Visualisera data
•Använd en eller flera lämpliga datareduktiontekniker
•Presentera data som en liten berättelse om data och variabler (5-10 bilder, 10 min presentation + 5 min frågor)

•EXTRA: 
•Gör interaktiv visualisering
•Undersök bidrag av olika viariabler
•Om ni använder UMAP använd hyperparameter justering
